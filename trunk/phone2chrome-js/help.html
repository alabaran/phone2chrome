Phone 2 Chrome Extension

> About

Phone 2 Chrome is a project consisting of a Chrome Extension, Android App, and using Dropbox service to send links from Android device to Chrome desktop browser.

> How does it works?

Android application will create a file in your Dropbox filesystem at:
/Dropbox/phone2chrome/pages
It contains information about links in standard json format. When you share a web link in your Android phone with Phone2Chrome app it will add a new entry to this file.
In your computer, you just need to launch Chrome and click on Phone2Chrome extension button. It will open each link in a new tab.
